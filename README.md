# 🔧 해킹 도구 설치 가이드

초보 해커들이 실습에 필요한 도구들을 손쉽게 설치하고 활용할 수 있도록 이 가이드를 제공합니다. 도구별 설명, 설치법, 주요 기능을 중심으로 정리했습니다.

---

## 📁 디렉토리 구조

- `CORE/`: 개발 환경 설정에 필요한 기본 도구들 (코드 에디터, 언어, 패키지 매니저 등)
- `Software hacking/`: 리버스 엔지니어링, 포렌식, 분석 도구 등 전문 해킹 도구 모음

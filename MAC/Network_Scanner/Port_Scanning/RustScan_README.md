# RustScan

## 설명
초고속 포트 스캐너로, Nmap과 연동하여 빠른 포트 검색을 지원합니다.

## 설치 영역
`/usr/local/bin/rustscan`

## 사용처
- 빠른 포트 검색
- 대규모 네트워크 스캔

## 설치(접속) 방법
1. **Homebrew를 통해 설치:**
   ```bash
   brew install rustscan
   ```
2. **설치 확인:**
   ```bash
   rustscan
   ```

# CURL

## 설명
커맨드라인 기반의 데이터 전송 도구로, HTTP, FTP 등 다양한 프로토콜을 지원합니다.

## 설치 영역
`/usr/bin/curl`

## 사용처
- HTTP 요청 테스트
- API 호출 및 디버깅
- 파일 다운로드 및 업로드

## 설치(접속) 방법
1. **macOS 기본 설치됨 (설치 확인):**
   ```bash
   curl --version
   ```
2. **최신 버전이 필요할 경우 Homebrew로 설치:**
   ```bash
   brew install curl
   ```
